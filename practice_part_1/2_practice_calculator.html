Alpine.js challenge: Simple Calculator See lecture on GitHub

Objectives:
  1. The 2 numbers in the left and right input are "connected" with the chosen operator in the middle.
  2. The result is shown underneath where it says "Result:".
  3. Display (and calculation) only occurs when the 3 fields have non empty value.
  4. Reset inputs, selcect and result with reset button.


Instructions

  1. Give the surrounding section a x-data, an object with the 3 keys num1, num2 and operator
    All those keys have a default of an empty string.
  2. Connect the 2 inputs + the select with x-model to num1, num2 and operator.
  3. Display the result with x-text after "Result" by using concatenation.
    Use eval(num1 + operator + num2) to calculate the result on the fly.
  4. Make sure that no error can occur, by putting the span for the result AND (seperately) the span for the Reset button each inside of a template tag. In those template tags use x-if with the condition:
    num1 != '' && num2 != '' && operator != ''.
  5. Use the reset button with @click to reset the inputs and the select and the result object.
