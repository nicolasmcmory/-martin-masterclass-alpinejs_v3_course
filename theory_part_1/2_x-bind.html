x-bind: Sets value of HTML attribute, e.g. value, style, class etc."

x-bind (:) - manipulate HTML attributes

Example 1: Set value of input with object value

Input value is bound to "myVar"

Example 2: Set style attribute

Some text with some font weight
font-weight is set by "myStyle"

Example 3: Set class attribute

background-color is set by "myClass"

Steps Example 1

  1. In Input define x-data="{ myVar: 'new text'}" and then bind: :value="myVar"

Steps Example 2

  1. In Span define x-data="{ myStyle : 'font-weight:700'}" and then bind: :style="myStyle"

Steps Example 3

  1. In Div define x-data="{ myClass : 'bg-blue-500'}" and then bind: :class="myClass"

Things to remember:

  • Always use a colon (:) instead of x-bind - it's easier!
