x-model: The Value of element(s) and the value of an object (data) influence each other. This is 2-Way-Binding!

x-model: set object value to input value

Example 1:Text in Input element goes to output immediately

Example 2:Text in Inputs can be influenced by both inputs

Example 3: Color of Div AND Select is influenced by chosen color.

Take Away:

  • x-text and x-bind(:) can use concatenation, eg.:
    x-text="'Output: ' + myInput"
    :class="selectClasses + ' bg-' + myColor + '-600'"

Reference

  • Color system of Tailwind CSS

Steps Example 1

  1. Give the section x-data="{myInput : ''}"
  2. Give the span x-text="'Output: ' + myInput"
  3. Give the input x-model="myInput"

Steps Example 2

  1. Give the section x-data="{myValue : 'start'}"
  2. Give BOTH the inputs x-model="myValue"

Steps Example 3

  1. Give the section a x-data with myColor: 'gray',
  2. Add another key selectClasses: '' to this x-data
  3. Now cut out all the classes BUT the bg-gray-600 and put it into the value for the key selectClasses.
  4. The bg-gray-600 you can delete.
  5. Now bind the color of the div with: :class="'bg-' + myColor + '-600'"
  6. Now bind the color of the select with: :class="selectClasses + ' bg-' + myColor + '-600'"
  7. Connect the select value to the object "myColor" with x-model="myColor"
